# TOSSTOUCH Website

This is the official website for **TOSSTOUCH Hair Point**, built using **HTML**, **CSS**, and **JavaScript**.  
The project showcases a clean layout, responsive design, and simple navigation across all pages.

---

##  Project Structure

The project contains the following files and folders:

- `index.html` – Home page  
- `services.html` – Services page  
- `contact.html` – Contact page  
- `styles.css` – Main stylesheet  
- `script.js` – JavaScript file (if interactions are needed)  
- `assets/` – Folder containing images and any other media  
- `README.md` – Project documentation

##  About the Project

This project is a simple front-end website designed to present:

- A hero banner with text overlay  
- About section  
- Services offered  
- Contact form and business information  
- A responsive layout that adjusts to different screen sizes  

The purpose of this project is to learn and showcase front-end development skills.
